In python, anything that you enclose between single or double quotation marks is considered a string.<\/p>\n\n\n\n

A string is essentially a sequence or array of textual data. Strings are used when working with Unicode characters.<\/p>\n\n\n\n

Example<\/h2>\n\n\n\n
name = \"Exia\"\nprint(\"Hello, \" + name)<\/code><\/pre>\n\n\n\n
Output<\/h2>\n\n\n\n
Hello, Harry<\/p>\n\n\n\n
Note: It does not matter whether you enclose your strings in single or double quotes, the output remains the same.<\/p>\n\n\n\n
Sometimes, the user might need to put quotation marks in between the strings. Example, consider the sentence: He said, \u201cI want to eat an apple\u201d<\/strong>.<\/p>\n\n\n\n
How will you print this statement in python?: He said, \"I want to eat an apple\".<\/code> We will definitely use single quotes for our convenience<\/p>\n\n\n\n
print('He said, \"I want to eat an apple\".')<\/code><\/pre>\n\n\n\n
Multiline Strings<\/h2>\n\n\n\n
If our string has multiple lines, we can create them like this:<\/p>\n\n\n\n
a = \"\"\"Lorem ipsum dolor sit amet,\nconsectetur adipiscing elit,\nsed do eiusmod tempor incididunt\nut labore et dolore magna aliqua.\"\"\"\nprint(a)<\/code><\/pre>\n\n\n\n
Accessing Characters of a String<\/h2>\n\n\n\n
In Python, string is like an array of characters. We can access parts of string by using its index which starts from 0.
Square brackets can be used to access elements of the string.<\/p>\n\n\n\n
print(name[0])\nprint(name[1])<\/code><\/pre>\n\n\n\n
Looping through the string<\/h2>\n\n\n\n
We can loop through strings using a for loop like this:<\/p>\n\n\n\n
for character in name:\nprint(character)<\/code><\/pre>\n\n\n\n
